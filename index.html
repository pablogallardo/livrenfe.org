LivreNFE

O emissor de NFe código aberto para Linux

LivreNFE é um emissor de notas fiscais orientado para pequenas e médias empresas, código aberto, feito para linux, desenvolvido em C.

Este é o primeiro release do projeto. Está disponível para fazer testes e sugerir melhorias. Só é possível emitir notas fiscais eletrônicas em homologação

Características

  • Grátis: A ferramenta é opensource, ou seja, o código está disponível no GitHub, qualquer pessoa pode fazer download e usá-la.
  • Simplificada: Este emissor que está orientado para pequenas e médias empresas é facil de usar e é possível emitir notas fiscais rapidamente.
  • Customizável: Ao ser código aberto, é possível implementar funcionalidades específicas para a sua empresa.

Como fazer sugestões ou reportar erros

A sua opinião é muito importante e suas sugestões podem ser implementadas no projeto, para fazer uma sugestão ou reportar error, é necessário abrir Issues no GitHub ou enviar um e-mail para pggllrd@gmail.com
